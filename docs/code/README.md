# Python Code Modules

Production-ready implementations of AI marketing systems.

## Installation

```bash
pip install -r requirements.txt
```

## Modules

- **segmentation.py** - Customer behavioral clustering
- **churn_prediction.py** - Churn risk scoring
- **personalization.py** - Hybrid recommendation engine
- **bias_detection.py** - AI ethics & fairness auditing

See individual files for documentation.
