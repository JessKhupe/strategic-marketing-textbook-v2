# Chapter 13: Creative Systems & Generative Content

!!! note "Bloom's Level & Duration"
    **Bloom's Level:** Apply, Create  
    **Duration:** 2 weeks

## Summary

Explores how generative models (text, image, audio, video) integrate into creative workflows to produce variant libraries, automate personalization, and enable rapid testing.

## Learning Objectives

1. Build a prompt-to-asset pipeline for copy and image generation.  
2. Implement versioning, review, and human-in-the-loop checkpoints.  
3. Automate multivariate creative testing using generated variants.

## Frameworks

| Framework | Purpose |
|---|---|
| Prompt‑Ops | Standardize prompts and evaluation metrics |
| Asset Versioning | Track lineage of generated content |

## Tools

| Tool | Use case |
|---|---|
| OpenAI / Anthropic | Text generation and summarization |
| Midjourney / DALL·E | Image generation for creative variants |
| Descript / ElevenLabs | Audio and voice cloning for ads |

## Ethics & Brand Safety

- Ensure generated content adheres to brand voice and legal standards.  
- Implement classifiers to filter hallucinations, bias, and unsafe outputs.

## Exercise

Generate 12 ad variants for a single product using controlled prompts, run a simulated A/B test, and analyze performance.
