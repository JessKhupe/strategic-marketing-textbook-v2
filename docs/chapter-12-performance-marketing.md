# Chapter 12: Performance Marketing & Measurement

!!! note "Bloom's Level & Duration"
    **Bloom's Level:** Apply, Analyze  
    **Duration:** 2 weeks

## Summary

This chapter covers paid media strategy, full-funnel performance marketing, and measurement techniques including experiments, incrementality testing, and cross‑device attribution.

## Learning Objectives

1. Plan full‑funnel paid media programs with CPA/CAC targets.  
2. Design incrementality tests and interpret uplift results.  
3. Instrument events and use event-driven analytics for optimization.  
4. Implement automated bid strategies with guardrails.

## Core Topics

- Audience selection and lookalike strategies
- Creative testing workflows (MVT, bandit algorithms)
- Measurement: experiments, holdout groups, Geo experiments
- Budget pacing and bid automation

## Tools & Patterns

| Tool | Purpose |
|---|---|
| Google Ads / Meta Ads | Campaign execution and automation |
| Snowflake / BigQuery | Store event-level data for analysis |
| DSPs / Programmatic | Reach and frequency control |

## Practical Activity

Students design an incrementality experiment for a seasonal campaign and prototype a monitoring dashboard to track key metrics.
